Demo Narrative

1. Sign up → Build your video profile

Candidates register, complete KYC identity verification, upload or record a 60-second video pitch, and submit with skills and resume.

2. Discover jobs → Apply in one click

Browse open positions, see required skills, apply instantly. A recruiter conversation is auto-created with a welcome message.

3. AI ranks candidates for recruiters

6-factor AI scoring: skill match, semantic fit, experience level, video presence, profile completeness, KYC status.

4. Drag pipeline → Schedule → Chat

Drag candidates across Applied → Shortlisted → Interview → Offer. Schedule interviews with Google Calendar & .ics export. Message candidates directly.

How VidHire Works

1

Candidate Profile

Captures identity, role, experience, skills, portfolio, resume, and video pitch.

2

AI Signal Layer

Turns profile data into summaries, match scores, fit reasons, and interview questions.

3

Recruiter Workspace

Lets recruiters post jobs, search candidates, compare match quality, and review video context.

4

Hiring Pipeline

Moves candidates from Applied to Shortlisted, Interview, and Offer for an ATS-style demo.

Candidate Features

Authentication — role-based login/signup (candidate & recruiter)

KYC — 3-step identity verification with selfie capture

Video Resume — browser recording via MediaRecorder API or file upload

Job Discovery — browse, search, one-click apply with auto conversation

Application Tracking — live status with color-coded stages

Recruiter Features

AI Matching — 6-factor weighted scoring with semantic skill grouping

Resume Parsing — NLP keyword extraction from resume text

Kanban Pipeline — drag-and-drop across Applied → Shortlisted → Interview → Offer

Calendar Sync — Google Calendar deep-link + .ics download on every interview

Messaging — real-time recruiter ↔ candidate chat

Tech Stack

Frontend — Vanilla JS, HTML5, CSS3, MediaRecorder API, Drag & Drop API

Backend — Node.js built-in HTTP server, zero dependencies

Storage — JSON file (server) + localStorage (client fallback)

Calendar — RFC 5545 iCal format + Google Calendar URL API

AI — Multi-factor weighted algorithm with semantic skill categories
